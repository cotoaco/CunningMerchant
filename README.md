# CunningMerchant
Программа для расчета оптимальной расстановки тюков на корабле во время шторма
## структура файлов
```
list.h - заголовычный файл с описанием односвязного списка  
checker.h - заголовочный файл для класа проверок
logic.h - объявления класса решения
logic.cpp - реализация алгоритма решения
main.cpp - точка входа и меню
```
## компиляция
`g++ main.cpp logic.cpp -o main`
## запуск
` ./main`
